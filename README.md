# AI on command-line

Firstly, configure your secret_key on the project. 

## How it works:

```bash
tiago@MacBook-Air-de-Tiago ~ % ai
User Input:
> Olá Gemini


IA output:
\Saudações! Como posso ajudar?


User Input:
...
```

To run `ai` command in your CLI, run the command bellow in the project on root dir:
```bash
npm install -g .
```
